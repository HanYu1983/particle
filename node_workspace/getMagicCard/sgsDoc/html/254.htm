魔王之脑·李儒
费用： 3 势力需求：
武/体： 2/3
势力： 群 类别： 无双单位 ~ 军师
插画作者： 孟迭 系列名称： 群星 卡牌编号： 81
技能描述

当李儒进战场时，你进行一次判定，若判定牌的打出费用大于或等于3，则每位对手弃一张牌；若否，则每位玩家弃一张牌。
每当任一对手弃一张牌时，除非其牺牲两个领土，否则失去3点体力。

“逼他服毒？不，这太不优雅。”

单卡解惑

2015-07-24：李儒的第一能力中，如果判定牌的打出费用小于3，进攻玩家先选择一张手牌面朝下，防御玩家再选择一张手牌面朝下，然后双方一起展示并弃掉该牌。

2015-07-24：李儒的第二能力结算时，对手必须选择其一执行，如果该玩家只有一个领土，他便只能选择失去3点体力。
